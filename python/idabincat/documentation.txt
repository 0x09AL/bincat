


BinCAT:
L'outil BinCAT se présente sous la forme :
+ d'un plugin IDA écrit en python 
+ du module analyseur 

Le plugin IDA peut être chargé depuis l'interpreteur python d'IDA après avoir copié les fichiers de "BinCAT" dans le répertoire python d'IDA (ex: /opt/ida6.7/python).


+ analyzer.py 

+ La version Plugin
